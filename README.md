# 2048

A clean in-game version of 2048 playable in Geometry Dash!

## What it does

- Adds a 2048 launcher button to the menu
- Opens a dedicated menu with the full 4x4 board
- Supports arrow keys and WASD
- Detects both the 2048 win state and game over

## Notes

Controls stay contained to the popup and the game can continue even after hitting 2048.

## Controls

- Arrow Keys / WASD to move
- R to restart instantly
