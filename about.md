# 2048

A clean in-game version of <cy>2048</c> playable in <cg>Geometry Dash!</c>

## What it does

- Adds a <cy>2048</c> launcher button to the menu
- Opens a dedicated menu with the full <co>4x4 board</c>
- Supports <cg>arrow keys</c> and <cg>WASD</c>
- Detects both the <cg>2048 win state</c> and <cr>game over</c>

## Notes

Controls stay contained to the popup and the game can continue even after hitting <cy>2048</c>.

## Controls

- <cg>Arrow Keys</c> / <cg>WASD</c> to move
- <co>R</c> to restart instantly
